A Evolução da Internet e da Web

A jornada tecnológica da sociedade moderna se confunde com o surgimento das redes de computadores. De experimentos militares a uma ferramenta global de comunicação, a evolução foi exponencial.

Linha do Tempo Textual

  1. Surgimento das redes: Primeiras ideias de conexão entre máquinas distintas nos anos 60.
  2. ARPANET: A primeira rede de comutação de pacotes, financiada pelo Departamento de Defesa dos EUA.
  3. TCP/IP: Protocolo estabelecido nos anos 80 que unificou as diversas redes em uma única "Internet".
  4. DNS: Criação do Sistema de Nomes de Domínio, substituindo a necessidade de decorar endereços IP.
  5. World Wide Web: Sistema de informação criado no CERN focado em hipertexto.
  6. Tim Berners-Lee: Cientista britânico inventor da Web.
  7. HTML: Linguagem de marcação criada para formatar documentos na rede.
  8. HTTP: Protocolo fundamental para transferência dos documentos hipertexto.
  9. Navegador: Software cliente capaz de renderizar o HTML (ex: Mosaic, o primeiro popular).
  10. Servidor Web: Máquina responsável por hospedar e entregar as páginas aos clientes.

Da Web 1.0 às Aplicações Modernas

A Web 1.0 era predominantemente de "leitura", composta por páginas estáticas institucionais. As aplicações Web modernas oferecem interatividade avançada, execução de scripts no cliente, comunicação assíncrona e arquiteturas orientadas a serviços.

Estrutura Base

Tabela de Marcos Históricos

Principais acontecimentos da Internet e Web
Ano Acontecimento Importância para a Web ou Internet
1969 Primeira mensagem na ARPANET Provou que a comunicação em rede remota era viável.
1983 Transição para o TCP/IP Padronizou a comunicação, criando a infraestrutura da Internet moderna.
1989 Proposta da Web no CERN Iniciou o projeto de compartilhamento de documentos via hipertexto.
1990 Primeiro Servidor e Navegador Transformou o projeto em realidade prática utilizável.
1993 Lançamento do navegador Mosaic Trouxe interface gráfica, popularizando o acesso à Web pelo público geral.
1994 Fundação do W3C Garantiu padrões abertos e consistentes para o desenvolvimento futuro da rede.
Fim do escopo cronológico inicial.
