Lista de Exercícios 2 - CSS

Atividade 1 – O que é uma regra CSS?

Uma regra CSS é a estrutura básica utilizada para aplicar estilo a um ou mais elementos HTML. Ela determina "quem" receberá o estilo e "o que" será alterado. Uma regra é composta por:

  • Seletor: Indica qual elemento HTML será estilizado (ex: uma tag, classe ou ID).
  • Declaração: Fica dentro de chaves {} e contém a propriedade e o valor.

Exemplo prático:

h1 {
    color: blue;
    font-size: 24px;
}

Atividade 3 – Como incluir CSS no HTML?

Podemos incluir CSS em documentos HTML de três maneiras principais:

  1. Inline (Em linha): Usando o atributo style diretamente na tag HTML.
  2. Interno (Incorporado): Usando a tag <style> dentro da seção <head> do documento.
  3. Externo: Criando um arquivo .css separado e vinculando-o ao HTML usando a tag <link rel="stylesheet" href="estilo.css">.
